
# 🚀 Prept AI — Full Stack AI Interview Marketplace SaaS

An AI-powered interview preparation and mock interview marketplace platform built using modern full-stack technologies including React 19, Next.js 16, Supabase, Stream Video SDK, Google Authentication, and Tailwind CSS.

This project enables users to practice AI-driven mock interviews, schedule live video interview sessions, chat with interviewers, and improve their technical interview skills in a scalable SaaS environment.

# ⚙️ Environment Variables

Create a `.env.local` file and add the following variables:

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

NEXT_PUBLIC_STREAM_API_KEY=
STREAM_SECRET_KEY=

ARCJET_KEY=
```

# 🚀 Getting Started

## 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/interviewiq-ai.git
```

## 2️⃣ Navigate into the project

```bash
cd interviewiq-ai
```

## 3️⃣ Install dependencies

```bash
npm install
```

## 4️⃣ Run the development server

```bash
npm run dev
```
